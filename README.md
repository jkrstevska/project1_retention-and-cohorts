# Project 1: Retention & Cohorts

## Business Objective:
This project simulates a subscription-based digital product where user retention is a key driver of long-term revenue. 

## Problem Statement:
The business wants to understand how well users are retained over time and identify where engagement drops after sign-up.

## Key Questions
1. What percentage of users return in subsequent months after signup?
2. How does retention differ across signup cohorts?
3. When does the largest drop in user activity occur?

## Dataset:
Using a public BigQuery dataset
```bigquery-public-data.google_analytics_sample.ga_sessions_*```
